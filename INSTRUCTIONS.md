# Инструкции за стартиране

## Стъпка 1: Инсталация на зависимости

Уверете се, че имате инсталиран:
- .NET 8.0 SDK
- SQL Server LocalDB (или SQL Server Express)

## Стъпка 2: Възстановяване на пакетите

Отворете терминал в папката на проекта и изпълнете:

```bash
dotnet restore
```

## Стъпка 3: Създаване на базата данни

Изпълнете следните команди за създаване на миграциите и базата данни:

```bash
dotnet ef migrations add InitialCreate
dotnet ef database update
```

**Забележка:** Ако нямате инсталиран Entity Framework Core Tools глобално, изпълнете първо:
```bash
dotnet tool install --global dotnet-ef
```

## Стъпка 4: Стартиране на приложението

```bash
dotnet run
```

Приложението ще стартира на `https://localhost:5001` или `http://localhost:5000`.

## Стъпка 5: Вход като администратор

Използвайте следните данни за вход:
- **Имейл:** admin@hotel.com
- **Парола:** Admin123!

## Функционалности

### За потребители:
1. Регистрация на нов акаунт
2. Преглед на каталога със стаи
3. Резервация на стая
4. Симулирано плащане
5. Преглед на собствените резервации

### За администратори:
1. Управление на стаи (добавяне, редактиране, изтриване)
2. Управление на резервации
3. Преглед на отчети за заетост и приходи

## Структура на базата данни

Базата данни се създава автоматично при първото стартиране. Съдържа:
- Таблици за потребители (ASP.NET Core Identity)
- Таблици за стаи (Rooms)
- Таблици за резервации (Bookings)
- Таблици за плащания (Payments)

## Забележки

- Снимките на стаите се съхраняват в `wwwroot/images/rooms/`
- Плащанията са симулирани - няма реални транзакции
- При първото стартиране се създават 5 примерни стаи
