# Room Images

## Структура на папките

- `/images/` - Снимки за примерните стаи (seed data)
- `/images/rooms/` - Снимки, качени от админ панела

## Необходими снимки за seed data

За да работят примерните стаи, трябва да добавите следните снимки в тази папка:

1. `room-single.jpg` - Снимка за единична стая
2. `room-double.jpg` - Снимка за двойна стая  
3. `room-suite.jpg` - Снимка за суит стая
4. `room-family.jpg` - Снимка за семейна стая

## Откъде да вземете снимки

### Безплатни източници:
- **Unsplash**: https://unsplash.com/s/photos/hotel-room
- **Pexels**: https://www.pexels.com/search/hotel-room/
- **Pixabay**: https://pixabay.com/images/search/hotel%20room/

### Препоръчителни търсения:
- "hotel room single"
- "hotel room double bed"
- "luxury hotel suite"
- "family hotel room"

## Размери на снимките

Препоръчителни размери:
- Ширина: 800-1200px
- Височина: 600-800px
- Формат: JPG или PNG
- Размер на файла: под 500KB за по-бързо зареждане

## Как да добавите снимки

1. Изтеглете снимки от безплатни източници
2. Преименувайте ги според списъка по-горе
3. Поставете ги в тази папка (`wwwroot/images/`)
4. Рестартирайте приложението
