# Система за управление на хотел/къща за гости

ASP.NET Core MVC приложение за управление на хотел с възможности за онлайн резервации.

## Функционалности

- ✅ Регистрация и профили на потребители
- ✅ Каталог със стаи (снимки, цени, наличност)
- ✅ Онлайн резервация и плащане (симулирано)
- ✅ Админ панел за добавяне и редакция на стаи
- ✅ Отчети за заетост и приходи

## Технологии

- ASP.NET Core 8.0 MVC
- Entity Framework Core
- ASP.NET Core Identity
- SQL Server (LocalDB)
- Bootstrap 5

## Инсталация

1. Уверете се, че имате инсталиран .NET 8.0 SDK
2. Клонирайте или изтеглете проекта
3. Отворете терминала в папката на проекта
4. Изпълнете следните команди:

```bash
dotnet restore
dotnet ef migrations add InitialCreate
dotnet ef database update
dotnet run
```

## Първоначални данни

След първото стартиране на приложението се създава администраторски акаунт:

- **Имейл:** admin@hotel.com
- **Парола:** Admin123!

## Структура на проекта

- `Models/` - Модели на данни (Room, Booking, Payment, ApplicationUser)
- `Controllers/` - Контролери (Home, Account, Rooms, Booking, Admin)
- `Views/` - Razor изгледи
- `Data/` - DbContext и SeedData
- `wwwroot/` - Статични файлове (CSS, JS, изображения)

## Използване

### За потребители:
1. Регистрирайте се или влезте в системата
2. Прегледайте каталога със стаи
3. Изберете стая и направете резервация
4. Изберете метод на плащане (симулирано)
5. Получете потвърждение за резервацията

### За администратори:
1. Влезте с администраторския акаунт
2. Отворете админ панела
3. Добавете или редактирайте стаи
4. Управлявайте резервации
5. Прегледайте отчети за заетост и приходи

## База данни

Приложението използва SQL Server LocalDB по подразбиране. Можете да промените connection string в `appsettings.json`.
